# Phase 3: Growth Engine (June-July 2026)

## Goal
First 50 paying customers, A$5K MRR

## Sub-Goals

### 3.1 SEO Landing Pages [P0]
- Create dedicated landing page for each free tool (8 pages)
- Meta tags optimized for "startup dilution calculator australia" etc.
- Each page: tool + CTA to get full SVI analysis
- Target: 1,000 organic visits/month

### 3.2 Referral Program [P0]
- Earn 2 credits per successful invite
- Referral link on dashboard + post-analysis
- Track referrals in DB (referrer_id on app_users)
- Target: 20% of new users from referrals

### 3.3 Evidence Auto-Rescore [P0]
- When user uploads evidence → trigger SVI re-analysis
- Show delta notification: "+5 SVI this week"
- Update svi_snapshots automatically
- Target: 3x evidence uploads per paid user

### 3.4 OAuth Connectors [P1]
- GitHub OAuth → commit count, stars, activity → PTD confidence 75%
- Google Analytics → sessions, users → TRE confidence 75%
- Stripe OAuth → MRR, churn → TRE confidence 90%
- Target: 30% of paid users connect 1+ source

### 3.5 Email Sequence Automation [P0]
- Post-free-analysis: 6 emails over 14 days
- Post-payment: 7 emails over 30 days
- Churn prevention: 4 emails on cancel trigger
- Target: 10% email→conversion rate

### 3.6 Milestone Badges [P1]
- 15+ badges: first_analysis, evidence_uploaded, svi_100, investor_shared, etc.
- Badge engine + notification
- Visible on profile + dashboard
- Target: 80% of users earn 3+ badges

### 3.7 PDF Export [P1]
- Export 10-page SVI report as PDF
- Branded with logo + footer
- Shareable via email + Drive
- Target: 50% of paid users export PDF

### 3.8 Homepage A/B Testing [P1]
- Test 5 headline variants (from PRD Section 17)
- Track conversion rate per variant
- Auto-select winner after 1,000 impressions
- Target: +20% conversion improvement

### 3.9 Accelerator Dashboard [P1]
- Cohort-level SVI tracking
- Bulk founder onboarding
- White-label branding option
- Target: 2 accelerator partners

### 3.10 Pricing Optimization [P1]
- A/B test $1 vs $5 vs $25 per analysis
- Test credit pack sizes
- R&D Agent weekly pricing review
- Target: Maximize revenue per user
