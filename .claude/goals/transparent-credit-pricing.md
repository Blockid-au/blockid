# Master Goal — Transparent Credit Pricing & Modular Report System

## Vision
Every credit charge must be transparent, predictable, and shown to the user BEFORE execution. Users choose exactly what they want analyzed, how deep (word count), and see the cost upfront. The pricing must feel fair — significantly cheaper than hiring a consultant or using generic AI assistants that lack real startup data. All financial, legal, and business analysis must comply with Australian standards.

## Core Principles

### 1. Transparency First
- Show exact credit cost BEFORE any AI analysis runs
- Break down cost per section so user can cherry-pick
- Display estimated word count / depth for each price point
- Never charge without explicit user consent ("Analyze for X credits?")
- Show running total if user selects multiple sections

### 2. Micro-Pricing Psychology
Users pay small amounts per action, not one large sum. This reduces friction:
- A$0.10-0.50 per quick scan feels like "nothing"
- A$0.75-1.50 per deep section feels like "a coffee"
- A$2-5 for a full report feels like "a lunch"
- Compare: Consultant charges A$200-500/hour for the same analysis
- Compare: Generic ChatGPT/Claude has no real startup data, no SVI scoring, no data room integration

### 3. Value Anchoring
Show the user what they GET, not just what they PAY:
- "100-word scan: Quick signals check — 0.10 credits (A$0.10)"
- "500-word analysis: Detailed findings + gaps — 0.50 credits (A$0.50)"
- "1000-word deep dive: Benchmarks + recommendations — 1.00 credit (A$1)"
- "2000+ word expert report: Consultant-grade with financials — 2.00 credits (A$2)"
- Anchor against: "A startup consultant charges A$300+ for this analysis"

### 4. Australian Compliance (MANDATORY)
All analysis must comply with:
- **ASIC**: Australian Securities & Investments Commission regulations
- **ACL**: Australian Consumer Law — no misleading claims about valuations
- **Privacy Act 1988**: Data handling for financial/personal information
- **GST**: All prices are GST-inclusive (Australian standard for B2C)
- **AFSL Disclaimer**: SVI is NOT financial advice, NOT a legal valuation
- **ESIC/ESVCLP**: Compliance with early-stage investor schemes
- **Fair Trading**: Transparent pricing, no hidden charges, refund rights

| **Standard** | ~500 | 0.50 | A$0.50 | Detailed analysis with gaps + recommendations |

### Pricing Formula
```
credits = base_rate × (target_words / 500)
```
Where:
- `base_rate` = 0.50 credits per 500 words (standard depth)
- Minimum charge: 0.10 credits (100 words)
- Maximum charge per section: 3.00 credits (unlimited words)

## New Agent: AU Compliance Officer

### Decision: YES — Create a new specialized agent

**Rationale**: Australian financial, legal, and business compliance is too specialized for any existing C-Level agent to handle as a side responsibility. A dedicated compliance agent ensures every report section, every pricing page, and every customer-facing analysis meets AU regulatory standards.

### Agent Definition

```
Agent: AU Compliance Officer
Skill: /au-compliance
Reports to: CFO (financial compliance) + COO (operational compliance)
```

**Responsibilities**:
1. Review every report template for ASIC/ACL/Privacy Act compliance
2. Ensure all pricing displays are GST-inclusive (AU Consumer Law)
3. Add appropriate disclaimers to financial/legal/investment sections
4. Verify ESIC/ESVCLP compliance for investor-related features
5. Audit credit charging for fairness (Australian Consumer Law)
6. Review terms of service and privacy policy for AU compliance
7. Ensure data handling meets Privacy Act 1988 + APPs
8. Monitor for changes in ASIC fintech guidance

**Dual-use**: The AU compliance agent serves both:
- **Internal**: Ensures BlockID platform complies with AU regulations
- **External**: Adds compliance context to customer reports (e.g., "ASIC requires X for your company structure")
