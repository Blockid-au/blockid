# Startup Valuation Expert Knowledge — BlockID C-Level Agent

## 5-Method Blend Framework (AU Pre-Seed)

### Method 1: Berkus Method
Assigns value to 5 risk-reduction factors (max A$200K each):
- Founder quality → A$0–200K
- Product/prototype → A$0–200K
- Market validation → A$0–200K
- Strategic relationships → A$0–200K
- Revenue evidence → A$0–200K
Max: A$1M pre-money

### Method 2: VC Method (Backwards)
```
PV = Exit Value / (1 + Target IRR)^years
Target IRR: 30-40% (AU early stage)
Post-money = PV; Pre-money = Post-money - Investment
```

### Method 3: Cost-to-Build
Sum of: code development + infrastructure + design + IP + team time
Typical pre-seed AU SaaS: A$50K–200K

### Method 4: Revenue Multiple (Discounted)
```
Value = ARR × Multiple × Discount_for_stage
Stage discount: 0.2× (pre-revenue), 0.5× (early), 1× (growth)
SaaS multiples: 5-10× ARR for early stage
```

### Method 5: Scorecard Method
Adjust regional median pre-money by weighted factors:
- Management team (30%)
- Market size (25%)
- Product (15%)
- Competitive environment (10%)
- Marketing channels (10%)
- Technology (5%)
- Other (5%)
AU pre-seed median: A$1.5M pre-money
