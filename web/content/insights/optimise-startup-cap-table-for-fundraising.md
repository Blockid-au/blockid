## Key Insight
A messy cap table can lead to:
- Dilution of ownership
- Confusion among investors
- Reduced valuation

| **Cap Table Mistakes** | **Consequences** |
| --- | --- |
| Unclear ownership structure | Investor confusion, reduced valuation |
| Inaccurate equity calculations | Dilution of ownership, disputes |
| Failure to update cap table | Outdated information, lost trust |

## What to do about it
Implement these best practices:
* Maintain accurate and up-to-date records
* Use a standardised cap table template
* Regularly review and update your cap table

## Optimising Your Cap Table
Consider these key factors:
* **Pre-money valuation**: Ensure accurate calculations
* **Equity splits**: Fairly distribute shares among founders and investors
* **Investor rights**: Clearly outline terms and conditions
