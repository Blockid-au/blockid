- **Pricing page:** Use testimonials 1 and 4 to show ROI at different price points (A$1 SVI analysis driving real outcomes).

- **Accelerator pitch:** Use testimonial 5 when pitching partnerships with accelerators and incubators.
- **Replace with real testimonials** as soon as genuine user feedback is collected. Prioritise video testimonials for highest-converting placements.

## Testimonial Collection Process

1. After a user completes their second SVI analysis (showing return usage), trigger an automated email asking for feedback.
2. Offer incentive: 2 free SVI credits for a written testimonial, 5 credits for a video testimonial.
3. Always get written permission before publishing. Use a simple release form.
4. Ask users to include specific numbers (SVI score, time saved, money saved) -- specificity converts.
5. Update this file quarterly as real testimonials replace templates.
